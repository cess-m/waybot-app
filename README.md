# Waybot - AI Math Tutoring Chatbot

Waybot is an AI-powered web application designed to support students in learning Calculus through guided, step-by-step problem solving. Instead of providing direct answers only, the chatbot is structured to help learners understand the reasoning process behind mathematical solutions.

## Overview

Waybot was developed as an educational chatbot for Calculus topics, including limits, differentiation, and integration. The system provides guided explanations, hints, and structured responses to help students work through problems more independently.

The project also includes teacher-facing functionality for monitoring student progress, as well as privacy and consent features for responsible student data handling.

## Key Features

* AI-assisted chatbot for guided Calculus problem solving
* Step-by-step explanations for mathematical procedures
* Hint-based support when students are unable to proceed
* Coverage of limits, differentiation, and integration
* Teacher portal for reviewing student progress
* Recent questions panel for tracking previous student interactions
* Student privacy and consent management features
