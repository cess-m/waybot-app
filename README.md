# Waybot — AI Math Tutoring Chatbot

An AI-powered web-based chatbot that helps students learn 
Calculus through step-by-step guided problem solving.

## About
Waybot was built to help students develop genuine understanding 
of math concepts instead of relying on AI for instant answers. 
It guides students through the solving process with hints and 
explanations.

## Features
- 🤖 AI chatbot for step-by-step Calculus explanations
- 📚 Covers Limits, Differentiation, and Integration
- 💡 Hints system when students get stuck
- 📊 Teacher Portal with student progress analytics
- 🔒 Student data privacy and consent management

## Tech Stack
- React + Vite (Frontend)
- JavaScript, HTML, CSS
- Node.js (Backend)
- AI API Integration
